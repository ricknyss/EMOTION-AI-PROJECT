# 😊 AI Face Emotion Detection System

## 🚀 Project Overview
This project is a real-time AI-based system that detects human emotions using facial expressions and provides smart recommendations based on the detected emotion.

The system uses computer vision and deep learning to analyze facial features and classify emotions such as Happy, Sad, Angry, Surprise, etc.

---

## 🎯 Features
- 🎥 Real-time face detection using OpenCV  
- 🧠 Emotion classification using CNN (TensorFlow/Keras)  
- 💡 Smart recommendations based on detected emotion  
- 🎨 Interactive UI built with Streamlit  
- 📊 Emotion history tracking  

---

## 🛠️ Tech Stack
- Python  
- OpenCV  
- TensorFlow / Keras  
- NumPy  
- Streamlit
